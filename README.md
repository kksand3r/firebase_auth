# 🔥 Firebase Auth App (Flutter + Firebase)

Це проект **Firebase Auth App**, розроблений на Flutter з використанням Firebase виключно для автентифікації користувачів. Додаток демонструє базові принципи роботи з Firebase Auth, включаючи реєстрацію, авторизацію та відновлення доступу.

🌐 **Жива демо-версія:** [https://flutter-auth-app-2f23c.web.app](https://flutter-auth-app-2f23c.web.app)

## 🚀 Основний функціонал

- **Автентифікація (Email/Password):** Реєстрація, вхід в систему та відновлення пароля реалізовано за допомогою пакету `firebase_auth`.
- **Захищений доступ:** Доступ до головного екрану (`HomeScreen`) можливий лише після успішної авторизації.
- **Маршрутизація стану:** Використовується `AuthWrapper` для автоматичного перенаправлення користувача на екран входу або на головний екран залежно від стану авторизації.
- **Екрани:**
  - 🔐 Екран входу (Login)
  - 📝 Екран реєстрації (Sign Up)
  - 🔄 Відновлення пароля (Forgot Password)
  - 🏠 Головний екран (Home Screen з можливістю виходу)

## 🛠 Технологічний стек

- **Flutter / Dart** (Фронтенд, UI та логіка)
- **Firebase Authentication** (Логін, реєстрація, управління сесіями)

## 📂 Структура проекту

Основна логіка знаходиться в папці `lib/`:
```text
lib/
  ├─ main.dart                 # Точка входу, ініціалізація Firebase
  ├─ firebase_options.dart     # Конфігурація Firebase
  ├─ screens/                  # Екрани додатку
  │   ├─ forgot_password_screen.dart
  │   ├─ home_screen.dart
  │   ├─ login_screen.dart
  │   └─ sign_up_screen.dart
  ├─ utils/                    # Допоміжні утиліти
  │   └─ auth_errors.dart      # Обробка помилок авторизації
  └─ widgets/                  # Віджети
      └─ auth_wrapper.dart     # Віджет для перевірки стану користувача
```

## 💻 Як запустити локально

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/kksand3r/firebase_auth.git
   ```
2. Перейдіть у папку проекту:
   ```bash
   cd firebase_auth
   ```
3. Встановіть залежності:
   ```bash
   flutter pub get
   ```
4. Запустіть додаток (на емуляторі, реальному пристрої або в браузері):
   ```bash
   flutter run
   ```

*Примітка: Проект вже містить базове підключення до Firebase (через `firebase_options.dart`), але для повноцінної роботи вам може знадобитися налаштувати власний проект Firebase за допомогою FlutterFire CLI.*
